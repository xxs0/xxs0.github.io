情人节谈情

发表于 2014-02-14   |   分类于 随想

今天，又有多少恋人带着幸福的笑容拿着两个红本走出民政局的大门！你说，他们都幸福吗？我说，是的，此刻他们是幸福的！那他们会一直这样幸福吗？我想，不会！那是为什么呢？…

爱情是美好的，从小到大的见闻，让我们相信着，也期待它的到来。但是在经历过以后，也许会发现爱情原来并未如想象中那样美，而是充满现实的。不过是柴米油盐酱醋茶，也有冲突、有背叛、有离异。当初的信誓旦旦、甜言蜜语可以化作争执、谩骂。多年积累的认知瞬间坍塌。会困惑，欲问苍天，为什么自己得不到想要的爱情？会偏执，爱情不过是一场虚幻而已。

爱情究竟是什么样的？我无法说清，最近在知乎上无意看到的一段回答觉得很好，分享一下。

【而立之年回望，感觉爱情都是幻想出来的，为什么从小的教育和电影要给这种幻想？没有幻灭就不会有伤害】

首先，男女之间的互相倾慕，痴迷，非血亲之间的互相信任，帮助，这是人类社会进步的基石，我们的许多法律和经济发展，都建立在这种说不清道不明的互相信任上，一开始，非完全功利性的合作让人类社会壮大和进步，让人类更有动力去思考除了生存安危，繁衍后代以外的生活广度和深度——在本能上，人肯定是向往爱情的，就好像古人看见鸟会飞也会向往一把一样，向往爱情和自由自在，是每个人心里一个模模糊糊的憧憬。

但其次，由于现在通讯技术的发达，沟通条件的便利和越来越高信息的散布效率，对爱情的各种营销的确过度了，爱情本身是一种各有标准的东西，普通人说不明白，以营销业绩为主力的商业机构更没有动力去说明白，于是这个时候，对爱情的宣传非常苍白，我们只看得见爱情美好的一面，比如说好像两个人非常轻易就互相信任，不离不弃，甜甜蜜蜜，白头偕老了……而这种把爱情简易化的宣传就好像一个高考状元说哦，我每天基本不复习，打打游戏也一样考进清华了嘛，广大平民百姓如果真信了，这个焦虑感马上上来了，为什么我读死读活也没有这个效果啊？是不是我特别不行啊？如果只有一个高考状元这么说，我们还能不信，但如果高考状元个个这么说呢，你身边的优等生个个看起来很轻松呢，这就容不得你信不信了。而我们现在的生活消费里的佼佼者品牌，哪一个不打爱情的主意？而这些品牌，哪一个没有实力做几个看起来很美的广告？于是正面的，你觉得有一颗戒指就是爱情，洒一瓶香水就是爱情，反面的，你觉得没有爱情，就是一个失败者，一个让亲人和朋友失望的卢瑟。而在各种各样的大众草根论坛，我们基本只对有“情节起伏”或者“不太容易重复的”经历更有阅读兴趣，如果我每天更新哦，今天我吃了一顿全家，然后洗洗睡了，肯定没人要看，所以为了吸引阅读量和提高传播效率，凡是说出来的生活充满了大量的裁剪，把爱情里平淡的重复，或者和单身时代同样存在的一些迷茫啊，难过啊，矛盾啊的情绪都给去除掉了，所以，你只看得见爱情和单身时代不同的5%，你看不见有爱情但其实和单身时代有许多情绪重合的95%。

这时候，当你充满信心去尝试爱情的时候，你其实大量品尝到到的是那个95%，你想妈的，被骗了，你觉得爱情是一种幻想，你不想干了。

当然你很幸运，你遇见了我，我可以帮你拨乱反正一下。

爱情这个东西，95%就是取决于你自己单身时代是个什么样的人。你如果懦弱，胆小，无聊，苍白，那么你的爱情，也一样懦弱，胆小，无聊，苍白，而且会因为多了一个人需要磨合，加剧了你生活中矛盾的出现频率。而如果你是一个勇敢，自信，乐于尝试，积极寻找各类解决办法的人，那么你的爱情也会一样提高你的信心，相信自己是对的，给自己带来积极的一面，而且，会因为多了一个人，互相信任和帮助，从而觉得生活更加美好。

爱情这玩意，遇强则强，遇弱则弱，而他所依靠的基础，就是你自己到底是个什么样的人，而他有一个功效，就是进一步残酷地认清自己，那些希望不劳而获的，往往会打着爱情的名义去寻觅不劳而获，以为这是爱情，那些胆小懦弱的，往往会在爱情的旗下寻求一个保护者，以为这是爱情，那些自卑自弃的，往往会在爱情的旗下继续自卑自弃，以为这是爱情……许多人，一旦脱离了血亲和社会规则，他们往往不知道怎么和人亲密接触，因为在没有法律和商业规则的模糊地带，许多人心里的贪婪，恐惧，迷惑，茫然都会纷纷现行，这不是爱情的力量，这是人类的本性，而你自己的人格是想滋养他，还是想克服他，都是可以的，爱情只是你人格决心里的一个催化剂。

你是什么样的人，就有什么样的爱情。如果你觉得人生皆苦，四大皆空，就这么一个人安静地过吧，那也不错。但是如果你相信人生应该战斗和努力，而爱情也是战斗和努力的一部分，哪怕最后一败涂地，但起码你不会看低自己曾经的战斗豪迈。那么你就去寻找你心仪的异性，陪他们喝酒，听他们说笑，他们烦恼的时候静静地握住他们的手或者为他们祈祷，快乐的时候和他们拥抱，外面狂风大浪，你也准备再坚持一下，你相信你一个人可以过的不错，但你也觉的，多一个人一起战斗，哪怕过一段时间他转身离开但起码在一起的时候很不错，反正也都是生活的味道，没有什么不好，而对于有些人，比如说我，除了尝尝生活的味道，也没有什么其他更重要的事情要做呀。

感谢作者@菠菜（@朱怡婧 ），她的其他问题的回答也不错。

如果把爱情作为一门功课的话，我觉得很多人可能还没及格。我不知道中国的父母们是怎么样了？上学时让孩子专心学习，把恋爱当作豺狼猛虎提防，一工作了态度马上180度大转变，催着子女快些结婚，又是相亲又是逼婚。说是了却牵挂，可是不知他们可曾想过，他的孩子们可能还是爱无能，因为从未有环境让他/她学习会爱的技能。让一群还没接触过训练的新兵奔赴生活的战场，不是很可笑吗？难道他们是想让孩子们在结婚之后再自学，这不是本末倒置了么？

总有人看着别人的美好故事幻想着一切也发生在自己身上，自己却不想付出一丝努力。但白头偕老，不离不弃的爱情不是一时冲动、更不是上天眷顾的结果，而是千百次争执的未放开双手。我常想，对于爱情我该怎么办？我想自己唯一能做的是提高自己，更好的学会爱一个人，学会沟通，学会尊重，学会责任和承担，有勇气一起面对生活的困难挫折，有闲心共享美好的幸福温暖，一起发现生活的乐趣。

在新的一年希望自己提高爱人的能力，为和你在一起的未来生活多拼搏一些。

祝所有的情人们…,元宵节快乐！^_^

给自己些改变

发表于 2013-04-26   |   分类于 生活

今天晚上，依旧登录空间和QQ扫视一下好友们的动态。看见学长在线，昨天他给我发短信告知他新号码，看他动态应该是换工作了，于是问了一下，可能当时不忙所以马上回复了。他说，“快毕业了吧”，我说是的，忙于毕业设计呢。然后和他聊了一下毕业设计的情况。他建议多去图书馆，和老师聊聊。我说感觉咱们专业做毕业设计都不积极呀，本意是想了解一下他们那时情况，谁知一顿教训，说“那你就积极啊，总是没心没肺，随波逐流，都老大不小了，等你毕业就知道什么是社会了”，顿时说得自己惭愧啊。还告诫我不要太内向，我知道他说的都是希望我能变更好，看着他也在努力的工作，听着他的话让我很有触动。

看到另外一个很久没联系过的朋友也在线，就和他发了消息。由毕业的事谈到最近的状况。我半调侃问道，情感问题可以考虑一下了吧。他说“不想”，我说这个可以有了。他说“想以后一个人环游世界去”，我问“两个人一起去也挺好的？”，他说喜欢一个人。开始我以为他的环球旅行也就是一般的给自己放个假，几个月时间出去转转之类的。后来知道自己错了，我知道很多人都有想要去环游世界的愿望，但他却在做着。他说这是他从小的梦想，一直在为之奋斗，35岁开始起航，所以现在好好挣钱。他还在坚持学习各种语言，他说自己喜欢摄影，还在学洗照片，打算记录自己的行程，回来了写本书。听着他描述自己的梦想，我觉得他很幸福，说着他的打算，我也说了一些，他说会采纳我的建议。听着他讲这些挺愉快的，等着他实现自己的梦想！他说，人活着不仅仅为了恋爱，结婚，生小孩的。人活着是活的精彩，享受生活的，不然来这世上也太浪费了。我想他在实现自己的价值吧.

这两个谈话让我很感动，看着他们为之努力的奋斗也感染了我，打破了我大学最后的显得有些颓废的生活，让我呼吸到新鲜的空气。我觉得人是沉溺于惯性的。我们很喜欢熟悉的环境，不喜欢改变，哪怕是不好的。其实之前本来不太想去聊的，可能是很久没和别人联络以至于变得懒惰了，很庆幸自己打破了这个僵局，给自己一个窗口了解平常圈子（天天在一起的同学）之外的人。所以时常和朋友联系，打破你平常的生活，获取新的信息。多和其他朋友联络，和其他不同层次的人接触会让你有不同的感受。

由一首歌突发的联想

发表于 2013-03-14   |   分类于 生活

今天下午上网，忽然想起《我是歌手》中有人唱的<move like jagger>，然后在网上找来原版视频看看。我突然对这首歌有些好奇，于是在网上了解一下。原来这首歌来自美国流行摇滚乐团Maroon 5，发行于2011年7月，并且连续三周 Billboard Hot 100的榜首，销量也不错。但是我记得这首歌在国内熟知应该也就是去年七八月的时候，当时《中国好声音》有人唱过这首歌，后来就红了。还有的就是那首<我的歌声里>也是通过这种方式红起来的。可能这些歌刚出来也发行之初就被那些音乐爱好者所熟知，但是让普通大众获知乃至后来大街上随处可见应该还是缘于这个节目吧。疑惑的是，一首这么红的歌为什么发行一年多才在国内被广泛传播呢？还是通过这类选秀节目被大众知道？我想可能跟习惯有关吧，大部分人还是听华语流行歌曲比较多。另外为什么是选秀节目这种方式呢？我不太清楚国外是什么形式，应该有专门的唱片行业推广，自己对这方面不清楚，不过国内音乐产业发展也是一个困境之中。最后我觉得根本还在我们获取信息的途径太单一、传统媒体仍然是大多数人信息来源（CCTV之类你懂的），信息太闭塞、太落伍了。难怪有人说如果需要学习最新的知识，你需要看看国外的东西。

从google中国和百度的数据看出，出现两次高峰，分别在2012年9月和2013年2月，这应该就是《中国好声音》和《我是歌手》这两个红火的节目贡献的。当然从google和百度的数据对比出还是有些不同，百度是突然出，现搜索数据即在2012年9月，并且马上达到高峰，而google在2011年8月开始有数据，也就是歌曲发布不久便有搜索，中途还出现几次小高峰，这也可以反映使用两个产品的用户差别。

另一件事，傍晚的时候室友在放视频，并不时发出“这就是生活呀”的声声感慨，于是我也凑看了一眼，是个叫《为什么贫穷？》的纪录片，讲的关于中国教育。后来查询了一下，原来这个纪录片不止一集，这个关于中国教育的是第八集。之前这个制作团队还发布了《为什么要民主？》的纪录片，这些我都没看过，以后找来机会找看看吧。

说到这集《为什么贫穷？》，我没看完，还是挺感触的。这是教育问题，也是社会问题，牵扯了太多的个人和家庭，它反映的是如此真实、这才是国情，但它不显示在政府报告里。每个人生来就不平等，这是可以理解的，所以才需要一个体制，在制度下保证每个人权利的公平、平等，可惜的是这个也不属于我们这。这就像是一场比赛，本来每个人的起跑线都不一样，为什么TMD有的人还可以闯红灯不被处罚，有些人等着绿灯还被罚。还要被人傻屄的问道“你幸福吗？”，说得对，“中国问题其实就是几百个家庭的问题”，能有多复杂呢！最近路透社大手笔，花了一年多精力帮咱中国人理了理关系，请欣赏（梯子自备！）

我说了这么多，挺乱的，它们有关系吗？

雪

发表于 2013-01-04   |   分类于 随想

今天，下雪了。等我醒來時表哥早就上班去了，我收拾一下從他家出門。一路上飄著不大的雪花，我背著包沒帶帽子，在路上行走著。我時而走得快時而走得慢，有事正經的走著，有時又漫無目的的邊耍邊走，踩在沒有腳印的雪上，聽到咯咯的雪響聲，還偶爾用鞋在地上划動，我很喜歡這樣的感覺，看著匆匆行人，縮在一起的身子，我卻沒有“冷”的感覺，一路上還用手捧起一手雪花擰成一團，然後拋向遠方，雖然手凍得通紅，但我卻很自在。我知道肯定有人對我這行為感到發笑，其實我也要對他們笑笑，如果能有個人一起享受這一切那就再好不過了。這場雪，這程路讓我找到青春的感覺，做自己愛的事，享受那种感覺，不在乎別人的眼光，2013的第一場雪，來的真及時，真巧，真好。
我知道，今天的日子很特別，事情的發展太多變，昨天我們居然還有了點小糾紛，我沒有騙你，我說的都是真的。2013.1.4.我一直都惦記著你，不管以後會怎樣，你永遠在我心裡，你知道嗎？妞

梦

发表于 2012-11-27   |   分类于 随想

 

应该是今天，做了一个特别的梦！怎么特别，前面挺乱的，理不太清，后面的细节还依稀记得，它让我从梦中醒来，哭着醒来，平静一下看看时间，不到两点。

梦中先是比较没有逻辑的事情，后来来到一个地方，周围有些熟悉的人，突然，奶奶出现在面前，好像还患有“阿兹海默病”，她把一个“透明的小塑料袋”递给我，说着“凌崽，把这给你”，意思好像是把最后的有价值的东西留给我。看着她缓缓的将“袋子”递给我，听着她刚叫出我的名字，一种莫名的强烈感觉憋在心口，我歇斯底里地嚎啕大哭起来，就这她身旁，伴随著一种痛苦、惋惜、自责、感动的感受，夹杂在一起，眼泪也流了出来，就这样哭唤着，自己醒了过来，发现自己的泪水真的流了下来。

我醒来时就和那时梦里一样，很怕，怕奶奶离去，想起奶奶年事已高，想起现状，我自己还没有给她带来任何“享受”，觉得很对不住她老人家。庆幸的是它只是一个梦，今天上午给奶奶打回去一个电话，等了一会，许奶奶叫她来接的，听来好像她还在做“祷告”，听到许奶奶叫唤她时心里的石头才落了下来，过一会奶奶来了，问她好不好，她说很好，还是不停的叮嘱我在外面要注意身体。我问她几遍吃饭没有，可能电话声音不是很大，她还是重复着要我注意身体，还怕电话太贵了似的不想多说，最后她说听不太清楚我说的话，我说要她注意身体，下次在打给她，然后挂了电话。

在记下这些东西的时候我也是头脑混乱，我描述不清那种感受，我希望把它记下来，能够不时的提醒一下自己要关心一下亲人，真的好怕突然失去，才发现自己一直“忙得”忘了记起他们。
