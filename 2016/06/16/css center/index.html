CSS居中-完整指南

发表于 2016-06-16   |   分类于 Web前端

来源链接：Centering in CSS: A Complete Guide

水平方向

行内元素

.center-children {
text-align: center;
}

  • 对于inline, inline-block, inline-table, inline-flex等有效

块级元素

.center-me {
margin: 0 auto;
}

  • 无论你想要居中的块级元素或父元素的宽度是多少，都有效。
  • 不能float一个元素到中间 trick though

多个块级元素

  • 如果有两个或多个块级元素需要在一列水平居中，用一个不同的display类型。

    .inline-block-center div {
    display: inline-block;
    text-align: left;
    }
    .flex-center {
    display: flex;
    justify-content: center;
    }

垂直方向

CSS中垂直方向居中有一点点困难

单行

  • 有时inline/text 元素能垂直居中，只因为上下padding1相等。
  • 使用line－height 等于高度

    .center-text-trick {
    height: 100px;
    line-height: 100px;
    white-space: nowrap;
    ｝

多行

  • 相等的padding－top和padding－bottom也能给多行文本居中效果。

  • 如果无效，可以是用table cell

.center-table {
display: table;
height: 250px;
background: white;
width: 240px;
margin: 20px;
}
.center-table p {
display: table-cell;
margin: 0;
background: black;
color: white;
padding: 20px;
border: 10px solid white;
vertical-align: middle;
}

  • 如果table－like无效，可以是用flexbox，一个单flex-child能在flex－parent中轻易居中

.flex-center-vertically {
display: flex;
justify-content: center;
flex-direction: column;
height: 400px;
}

记住，只有如果父容器有一个fixed height（px, %,etc）才有意义

  • 如果以上都无效，可以使用“ghost element”技术，

.ghost-center {
position: relative;
}
.ghost-center::before {
content: “ “;
display: inline-block;
height: 100%;
width: 1%;
vertical-align: middle;
}
.ghost-center p {
display: inline-block;
vertical-align: middle;
}

块级元素

知道元素高度

.parent {
position: relative;
}
.child {
position: absolute;
top: 50%;
height: 100px;
margin-top: -50px; // account for padding and border if not using box-sizing: border-box;
}

元素未知高度

.parent {
position: relative;
}
.child {
position: absolute;
top: 50%;
transform: translateY(-50%);
}

使用flexbox

.parent {
display: flex;
flex-direction: column;
justify-content: center;
}

水平、垂直居中

元素 fixed width 和height

  • 在绝对定位在50%／50%处后设置负margin 等于宽或高的一半。它有很好的跨浏览器支持

.parent {
position: relative;
}
.child {
width: 300px;
height: 100px;
padding: 20px;
position: absolute;
top: 50%;
left: 50%;
margin: -70px 0 0 -170px;
}

元素有未知宽、高

  • 未知宽高，可是用transform属性和一个50%的负translate。

.parent {
position: relative;
}
.child {
position: absolute;
top: 50%;
left: 50%;
transform: translate(-50%, -50%);
}

使用flexbox

.parent {
display: flex;
justify-content: center;
align-items: center;
}
